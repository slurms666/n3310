Controls

  • Arrow keys — menu navigation / movement
  • Enter — select / confirm / primary action
  • Space — secondary action
  • Backspace / Escape — back / quit
  • R — restart current game
  • M — mute / unmute

How to use it

  1. Use the phone menu to open Play Games.
  2. Pick a game and press Enter.
  3. Read the short help overlay, then start playing.
  4. Press Esc any time to leave the game cleanly.
